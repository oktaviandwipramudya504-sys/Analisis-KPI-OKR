# Analisis KPI & OKR Dashboard
**Oktavian Dwi Pramudya**

## 1. Tujuan
Dashboard ini dibuat untuk memantau kinerja departemen secara real-time guna mencapai target perusahaan.

## 2. Fitur Utama
* Visualisasi interaktif menggunakan Plotly.
* Deteksi otomatis KPI "At Risk" (<90%) dan "Over-achiever" (>110%).

## 3. Teknologi
* Python
* Streamlit
* Pandas
* Plotly
