Analisis KPI & OKR Dashboard
Dashboard interaktif untuk memonitor kinerja departemen berdasarkan Key Performance Indicators (KPI) dan Objectives and Key Results (OKR).

Fitur Utama
Visualisasi Data Interaktif: Menggunakan Plotly untuk memantau pencapaian KPI secara real-time.

Analisis Otomatis: Sistem deteksi otomatis untuk mengidentifikasi departemen yang berada di zona "At Risk" (<90%) dan melampaui target (>110%).

Business Insight: Memberikan rekomendasi tindakan langsung bagi stakeholder.

Teknologi yang Digunakan
Python: Bahasa utama untuk analisis data.

Streamlit: Framework untuk membangun dashboard web interaktif.

Pandas: Untuk pengolahan dan manipulasi data.

Plotly: Untuk visualisasi data interaktif.

Cara Menjalankan
Clone repository ini.

Install library yang dibutuhkan: pip install -r requirements.txt
Jalankan dashboard: python -m streamlit run app.py
Setelah Anda memperbarui README.md, jangan lupa lakukan git add README.md, git commit -m "update readme", dan git push kembali.
